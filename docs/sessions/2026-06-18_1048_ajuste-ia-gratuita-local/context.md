# Contexto — Ajuste para IA Gratuita e Local

## Situação Inicial
O contexto técnico definia um modelo BYOK genérico, mas sem especificar alternativas gratuitas para o usuário em uso pessoal.

## Motivação
Garantir custo zero contínuo para o usuário final, selecionando motores de IA gratuitos (Gemini Free API) e locais (Ollama).

## Restrições
Dependência de chaves de API externas ou da instalação local do Ollama por parte do usuário.
