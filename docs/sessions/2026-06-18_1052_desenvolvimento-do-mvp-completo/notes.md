# Notas e Observações — Desenvolvimento do MVP Completo

## Insights e Aprendizados
- Utilizar a classe `.open` para controlar a opacidade e visibilidade dos modais permite transições CSS fluidas, mantendo o controle no JS limpo.
- Rodar o servidor estático via `cmd /c` contorna políticas de segurança de script do PowerShell em ambientes Windows.
