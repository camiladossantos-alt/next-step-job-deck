# Contexto — Desenvolvimento do MVP Completo

## Situação Inicial
O repositório possuía apenas o plano de especificações e contexto técnico atualizados nas sessões anteriores, sem arquivos de código de aplicação.

## Motivação
Construir a aplicação prática do JSOS para uso pessoal, consolidando todas as 6 features especificadas em um único portal SPA com custo zero.

## Restrições
A aplicação roda 100% no cliente (browser) para evitar custos de servidor, usando LocalStorage do navegador.
