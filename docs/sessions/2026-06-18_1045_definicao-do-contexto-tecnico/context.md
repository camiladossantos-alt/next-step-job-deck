# Contexto — Definição do Contexto Técnico

## Situação Inicial
O arquivo de contexto técnico do repositório continha apenas placeholders de templates vazios.

## Motivação
Definir diretrizes técnicas claras de baixo custo e alta velocidade de implementação para as features do JSOS MVP, garantindo o início do desenvolvimento de forma previsível e alinhada.

## Restrições
A premissa do usuário exige custo zero de infraestrutura e hospedagem e ferramentas fáceis para codificação imediata.
